ajaxPlayer v0.01

XML Audio / Video


Plays Media via HTML5 w/ A Flash Fallback Option.

HUI's Fluid Theme allows the Player to fit nicely anywhere you need it to. While allowing quick theme options and jquery-ui integration.

Both the Audio And Video Playlists are loaded via AJAX from an XML file.

The media plugin code is based on jPlayer.
